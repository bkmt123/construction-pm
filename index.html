☁ Airtable Sync

Connect to Airtable so your PM data stays in sync across your phone, laptop, and any device you use on the job.

One-time setup:
1. Go to airtable.com → free account
2. Create a Base (e.g. "PM Hub")
3. Add two fields to the default table:
  • StorageKey (Single line text)
  • StateJSON (Long text)
4. Go to airtable.com/create/tokens → create token with data.records:read + data.records:write scopes on your base
5. Base ID is in the URL: airtable.com/appXXXXXX/...
